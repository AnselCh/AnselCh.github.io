[筆記]Microsoft Graph API建立行事曆事件

2023-08-20
工作筆記
  • GraphAPI
  • Microsoft

建立應用程式

  1. 進入Azure Portal後左側選取應用程式註冊
  2. 點選API權限 -> 新增權限
  3. 依照官方給予Calendars.ReadWrite權限
  4. Azure Portal後左側選取企業應用程式 -> 選擇剛剛建立的應用後，將使用者加入應用裡才會獲得使用API的權限
  5. 附帶的建立json格式參考官方範例，若要同時加入多位參與者可參考以下內容
  "attendees": [
        {
            "emailAddress": {
                "address": "{受邀者1的Email}",
                "name": "{受邀者1的名子}"
            }
        },
        {
            "emailAddress": {
                "address": "{受邀者2的Email}",
                "name": "{受邀者1的名子}"
            },
            "type": "required"
        }
    ],
  "organizer": {
    "emailAddress": {
      "address": "{會議召集人的Email}",
      "name": "{會議召集人的名子}"
    }
  }
